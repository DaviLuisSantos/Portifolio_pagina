Gerador de Jogos de Loteria

Um programa de geração de números aleatórios para loterias brasileiras, desenvolvido em C# usando a plataforma console. Ele permite que o usuário escolha a loteria desejada, faça quantos jogos quiser e garanta que os números gerados não sejam repetidos em um mesmo sorteio.

Características principais:

  • Interface de linha de comando fácil de usar, onde o usuário pode escolher a loteria desejada e o número de jogos a serem gerados.
  • Geração de números aleatórios usando a classe Random do C#, garantindo que os números gerados sejam realmente aleatórios.
  • Validação dos números gerados para garantir que eles não sejam repetidos em um mesmo sorteio.
  • Possibilidade de salvar os jogos gerados em um arquivo de texto para uso futuro.
  • Possibilidade de gerar jogos com base em números escolhidos pelo usuário.
